Scoring Tests

Scoring tests involves scanning through a completed test and determining whether the answer is right or wrong. The degree to which it is right or wrong is defined by a number of points. Completely right gets the maximum number of points whereas completely wrong gets no points. The amount of points each problem is worth is defined on a per test basis. The score of a problem can not exceed the maximum number of points defined.

Organizing Tags

After many problems have been defined the number of tags may become quite large. Methods may be necessary to manage the tags and make them easier to search.

If there are tags which are only slightly different they could be merged together. This would mean altering the definition of the problem after it has been registered with the database.

The tags being viewed could be restricted. If problems for several classes were stored it might be desireable to restrict the view to only "math" and not "history".

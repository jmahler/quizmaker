Adding Problems (to the database)

Once a problem has been created and the output has been previewed (the section called “Creating Problems in Perl”) it can be added to the database.

Figure 3.4. Adding a problem.

user$ qzm-load.pl <file> [<file2> ...]

Usage example of qzm-load.pl used to load a problem.


One problem that may arise is when the user attempts to add a file more than once. If a unique identifer (UUID) has been identified for the problem it will not be added again. Otherwise the sytesm will assume the problem is new and add it like normal.
